    # 使用http-server开启一个本地服务器

    1. 下载http-server
    npm install http-server -g
    1. 开启http-server服务
    http-server -c-1
// 或者http-server
//只输入http-server的话，更新了代码后，页面不会同步更新

    1. 在url后+ /doc_name.后缀名，可以获取文件

    2. 关闭http-server服务 按快捷键CTRL-C 终端显示^Chttp-server stopped.即关闭服务成功。
