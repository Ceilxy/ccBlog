## Vuex状态管理

### 概念

Vuex是vue应用程序开发的专门用于**状态管理模式**。它采用**集中式储存**管理应用的组件状态，并以相应的规则保证状态以**可预测的方式**发生变化。

### 为什么要状态管理

- 单向数据流

- 多个组件共享状态
当我们需要多个组件共享状态的时候
所以隔离状态管理，以一个**全局单例模式**管理状态时，任何组件都可以获取状态或者触发行为，还可以保持状态间的独立性
